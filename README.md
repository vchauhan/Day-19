Took an AI-powered "Football Intelligence" deep dive this week — part data analysis, part personality test — and the results were oddly accurate.

The exercise combined three things I wouldn't normally put in the same room: historical World Cup data, a football IQ quiz, and a personality assessment mapped against Messi and Ronaldo's traits.

A few takeaways that stuck with me:

→ Scored 100/100 on the football IQ quiz — turns out "casual fan" was underselling it.
→ My decision-making style came back 71% Ronaldo, 68% Messi — calculated and efficiency-driven, but with a quieter, lead-by-example streak underneath.
→ Landed the "Relentless Competitor" archetype: driven by hard work, a dislike of losing, and wanting to be recognized as the best — not just told it.

What I found genuinely interesting wasn't the football trivia — it was how cleanly behavioral traits (decision-making under pressure, leadership style, risk tolerance) map onto performance patterns we already track in elite sport. The same lens applies well beyond football.

Small reminder that self-awareness tools don't have to be serious to be useful.
